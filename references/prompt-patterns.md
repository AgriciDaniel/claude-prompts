# Prompt Engineering Patterns

Proven patterns extracted from analysis of 2,500+ curated prompts.

## Core Structure Patterns

### 1. Cinematic Shot Pattern (most common)
```
[Subject] in [setting], [camera spec], [lighting], [mood], [film reference]
```
Example: "A lone samurai standing in a bamboo forest at dawn, shot on RED V-Raptor, 50mm anamorphic lens, volumetric god rays, moody and contemplative, Kodak Vision3 500T color science"

### 2. Editorial/Fashion Pattern
```
[Person description] wearing [outfit], [pose], [setting], [lighting setup], [publication style]
```
Example: "Elegant woman with sharp cheekbones wearing an oversized Balenciaga trench coat, looking over shoulder, marble staircase, dramatic split lighting, Vogue Italia editorial"

### 3. Product Photography Pattern
```
[Product] on [surface/setting], [lighting setup], [camera angle], [style reference]
```
Example: "Luxury perfume bottle on polished black marble, three-point studio lighting with rim light, 45-degree angle, clean minimalist Apple-style product photography"

### 4. Fantasy/Concept Art Pattern
```
[Character/Creature] in [fantastical setting], [art style], [detail modifiers], [atmosphere]
```
Example: "Ancient elven warrior with crystal armor in a bioluminescent cave, digital painting style, intricate filigree details, mystical ethereal atmosphere"

### 5. Video/Motion Pattern
```
[Camera movement]: [Subject action] in [setting]. [Lighting]. [Style]. [Duration/Speed].
```
Example: "Slow tracking shot: A dancer leaps across a rooftop at golden hour. Warm backlight creates silhouette. Cinematic 24fps. Slight slow motion."

## Modifier Libraries

### Lighting Modifiers
- golden hour, blue hour, magic hour
- Rembrandt lighting, split lighting, butterfly lighting
- rim light, backlight, volumetric light, god rays
- neon, bioluminescent, candlelight
- overcast soft light, harsh midday sun
- studio three-point lighting

### Camera/Lens Modifiers
- shot on Canon R5, Sony A7IV, RED V-Raptor, ARRI Alexa
- 24mm wide angle, 35mm, 50mm, 85mm portrait, 135mm telephoto
- f/1.2 shallow DOF, f/8 sharp, f/16 deep focus
- anamorphic lens flare, tilt-shift miniature
- macro, fisheye, telephoto compression

### Style Modifiers
- photorealistic, hyperrealistic, cinematic
- editorial, documentary, street photography
- oil painting, watercolor, digital art, concept art
- anime, pixar style, dreamworks, comic book
- retro, vintage, Y2K, vaporwave, cyberpunk
- minimalist, maximalist, brutalist

### Film Stock References
- Kodak Portra 400 (warm, natural skin tones)
- Kodak Ektar 100 (vivid, saturated colors)
- Fuji Velvia 50 (extreme saturation, landscapes)
- Ilford HP5 (classic B&W grain)
- Kodak Vision3 500T (cinematic, tungsten balanced)
- Kodak UltraMax 400 (everyday film look)

### Mood/Atmosphere
- moody, ethereal, dramatic, serene, intense
- nostalgic, melancholic, euphoric, mysterious
- gritty, clean, dreamy, surreal, raw

## Anti-Patterns (Common Mistakes)

1. **Conflicting styles**: "realistic cartoon" -- pick one
2. **Too many subjects**: Focus on ONE main subject
3. **Vague modifiers**: "nice" and "good" add nothing
4. **Ignoring aspect ratio**: Match AR to content type
5. **Kitchen sink**: Adding every modifier doesn't help
6. **Missing lighting**: Single biggest quality differentiator
